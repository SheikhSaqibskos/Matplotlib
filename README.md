Getting Started
Prerequisites

To run the provided script, ensure you have Python installed with the following dependencies:

Matplotlib: pip install matplotlib

NumPy: pip install numpy

Running the Code

Clone this repository:

git clone https://github.com/SheikhSaqibskos/Matplotlib


Run the Python script:

python matplotlib_summary.py


The script will generate various plots showing examples of different chart types and customizations.

Detailed Explanation

The script demonstrates the following plotting techniques:

Basic Line Plot
A simple line plot showing how to plot X and Y values.

Sine & Cosine Waves
Visualizing sine and cosine waves using np.linspace to generate smooth values.

Custom Line Styles
How to customize line styles, colors, and markers in plots for better visual appeal.

Scatter Plot
Creating scatter plots with different markers and customizing their appearance.

Bar Chart
Demonstrating how to plot a bar chart with categorical data.

Histogram
Plotting a histogram to visualize the distribution of random data points.

Subplots
How to place multiple plots within the same figure using subplots.

Axis Limits
Modifying axis limits to zoom into specific regions of the plot.

Annotations
Adding annotations, including text and arrows, to highlight points on a plot.

Styles (Themes)
Applying pre-defined Matplotlib styles (such as ggplot) to change the look of your plots.
